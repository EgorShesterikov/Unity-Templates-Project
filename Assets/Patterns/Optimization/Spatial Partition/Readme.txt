Есть набор объектов, каждый из которых обладает позицией в пространстве. Сохраняем объекты в пространственной структуре данных, орназинованной на основе их позиций. Эта структура данных позволяет вам эффективно запрашивать объекты, находящиеся возле указанной позиции. Когда позиция объекта изменяется, обновляем позиционную структуру данных и поиск можно продолжать.
