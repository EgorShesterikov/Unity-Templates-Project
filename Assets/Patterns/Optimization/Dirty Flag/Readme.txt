Набор первичных данных изменяется со временем. Набор вторичных данных вычисляется на основе с помощью ресурсоемкого процесса. "Грязный" флаг отслеживает рассинхронизация вторичных данных с первичными. Он устанавливается, когда первичные данные изменяются. Если флаг установлен, когда нам понадобились вторичные данные, они вычисляются и флаг снимается. В противном случае используется уже вычислительные вторичные данные.
